🧮 Python Calculator
A simple and user-friendly **Calculator** application built using **Python** and **Tkinter**. This project provides a graphical user interface (GUI) that allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.
📌 Features
➕ Addition
➖ Subtraction
✖️ Multiplication
➗ Division
🔢 Decimal number support
🧹 Clear button to reset input
⚠️ Basic error handling for invalid expressions
🖥️ Simple and responsive graphical user interface
🛠️ Technologies Used
**Python 3**
**Tkinter** (Python's built-in GUI library)

🚀 Getting Started
Prerequisites
Python 3.x installed on your computer.
You can download Python from:
https://www.python.org/downloads/
▶️ How to Run
1. Clone this repository:
git clone https://github.com/your-username/python-calculator.git
2. Navigate to the project folder:
cd python-calculator
3. Run the program:
python calculator.py
💻 How to Use
1. Launch the application.
2. Click the number and operator buttons to create a mathematical expression.
3. Press the **=** button to calculate the result.
4. Press **C** to clear the input field.
5. If an invalid expression is entered, the calculator displays **Error**.
